## Python Whatsapp Messenger Sender
This is a simple Python project source code that sends messages to specified senders from the code instantly. It uses the 'Pywhatkit' library from the Python module.

## Features
1. The source code sends messages instantly.
2. Sends messages via Whatsapp web.
3. Easier to use with the simple-few lines of code.

## Requirements
1. Python (2.7,3.0-3.3) versions.
2. 'Pywhatkit' Library tool.

## Installation dependancies
1. pip install pywhatkit tool.
2. 'Bash'.
3. Integrated Development Environment (IDE*) -Preffered.
4. Python debuggers in the IDE.

## Usage
Below are steps of how you use the source code to successfully send messages to the defined receipients Whatsapp account:
1. On the Phone_number section replace with the number you would want to send a message to and also a text message on the message section from the code.
   ie; ->phone_number = "+254748473***"
       ->message ="Hello Counsel Andrew, This is a Python text message from ##ScriptServant003"
  . Ensure the Phone_number starts with the country code.(+254 for kenya, +1 for USA).
2. Internet connection is required.
3. Ensure that you have opened your Whatsapp Web account on your browser.
4. Run the source code with the python debugger. Type 'python .\WhatsappPyhtonhack.py'. replace the saved python file name with yours for successful allocation and opening of the source code by the terminal.
5. During execution, the source code will open the browser with the Whatsapp web opened and send the text to the sender's phone number successfully.
